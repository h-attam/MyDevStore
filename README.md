# My Dev Store

Next.js 14 ile geliştirilmiş modern e-ticaret ve portfolio sitesi. E-kitap ve ürün satışı yapabilir, aynı zamanda kişisel portfolyo olarak kullanılabilir.

## Özellikler

**E-Ticaret Sistemi**
- Ürün listeleme ve detay sayfaları
- Sepet yönetimi (Supabase ile senkronize)
- Ürün arama ve filtreleme (kategori, fiyat aralığı)
 - Stripe ile ödeme entegrasyonu

**E-Kitap Satışı**
- E-kitap listeleme ve detay sayfaları


**Kullanıcı Yönetimi**
- Email ve Google ile giriş
- Form validation (Zod)

- **Sipariş Takibi**
- Kullanıcı sipariş geçmişi
- Sipariş durumu takibi
- Sipariş detayları

- **Portfolio**
- Hakkımda sayfası
- Beceriler, deneyimler ve projeler
- İletişim bilgileri

- **Responsive Tasarım**
- Mobil uyumlu
- Dark mode desteği
- Modern UI/UX

## Teknolojiler

- **Framework**: Next.js 14 (App Router)
- **Dil**: TypeScript
- **Styling**: Tailwind CSS
- **Validation**: Zod
